# Anaxor_E-Commerce_Webite

PHP Product Wishlist Web Application

Description:
This is a PHP full stack web application developed using PHP, MySQL, HTML, CSS, and JavaScript.
The application provides user authentication and a product wishlist feature.

Features:
- User registration and login system
- Product listing page
- Add products to wishlist
- Remove products from wishlist
- Secure database connection using db_connect module

Technologies Used:
- PHP
- MySQL
- HTML
- CSS
- JavaScript

Project Structure:
- index.php – Home page
- login.php – User login
- register.php – User registration
- product_list.php – Displays available products
- wishlist.php – Displays user wishlist
- add_wishlist.php – Add product to wishlist
- remove_wishlist.php – Remove product from wishlist
- db_connect.php – Database connection

How to Run:
1. Install XAMPP or any local server.
2. Place the project folder inside the htdocs directory.
3. Import the database SQL file into phpMyAdmin.
4. Update database credentials in db_connect.php.
5. Open the browser and run the project using localhost.
